# Project: Sentiment Analysis on Amazon Product Reviews

## 1. Problem Statement

This project aims to perform sentiment analysis on Amazon product reviews. The goal is to build a model that can accurately classify a review as either positive or negative based on its text content. This can provide valuable insights for both consumers and sellers on the Amazon platform.



## Dataset

The Amazon review dataset can be downloaded from [here](https://nijianmo.github.io/amazon/index.html). Download the zipped json file of the category of video games 5 core, which can be found under the title Small subsets for experimentation. Once the download is complete, extract the file.
